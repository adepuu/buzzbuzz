# BUZZ BUZZ

## Description

BuzzBuzz is a social network proof of concept that run on NestJS using serverless

## Installation

```bash
$ yarn add @vendia/serverless-express aws-lambda serverless-plugin-optimize serverless-offline
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
